Lenguaje SQL

Que es?

El lenguaje de consulta estructurada (SQL) es un lenguaje de programación para almacenar y procesar información en una base de datos relacional. Una base de datos relacional almacena información en forma de tabla, con filas y columnas que representan diferentes atributos de datos y las diversas relaciones entre los valores de datos. Puede usar las instrucciones SQL para almacenar, actualizar, eliminar, buscar y recuperar información de la base de datos. También puede usar SQL para mantener y optimizar el rendimiento de la base de datos.

  • Símbolos: SQL utiliza símbolos como: ; (Punto y coma): Se utiliza para separar sentencias SQL. Cada sentencia SQL debe finalizar con un punto y coma. <> (Diferente): Se utiliza para comparar si dos valores no son iguales en cláusulas WHERE.\ GROUP BY: Se utiliza para agrupar filas en función de los valores de una o más columnas. Se utiliza comúnmente con funciones de agregación como SUM, COUNT, AVG, etc. INSERT INTO: Se utiliza para insertar nuevos registros en una tabla. Por ejemplo, INSERT INTO tabla (columna1, columna2) VALUES (valor1, valor2) agrega un nuevo registro a "tabla". AS: Se utiliza para asignar alias a columnas o tablas en una consulta. Por ejemplo, SELECT columna AS alias FROM tabla permite referirse a "columna" como "alias" en la consulta.

  • Sintaxis: La sintaxis de SQL se basa en comandos y cláusulas que permiten realizar consultas, inserciones, actualizaciones y eliminaciones de datos.

  • Esqueleto de programa: Los programas en SQL consisten en sentencias que definen operaciones sobre tablas de bases de datos.

  • Aplicaciones: Para editar y ejecutar SQL, se pueden utilizar entornos de gestión de bases de datos como MySQL Workbench, SQL Server Management Studio, o interfaces web que permiten ejecutar consultas SQL.

  • Paradigma: SQL es un lenguaje de consulta de bases de datos y no se adhiere a un paradigma de programación en el sentido tradicional, ya que se centra en la manipulación de datos.

  • Ejemplo de consulta SQL (Selección de datos): 

      SELECT nombre, edad
FROM empleados
WHERE departamento = 'Ventas' AND salario > 30000;
