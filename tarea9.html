Lenguaje PHP

Que es?

PHP es el acrónimo recursivo de Hypertext Preprocessor, un lenguaje de programación de código abierto muy utilizado especialmente en el desarrollo de aplicaciones y web, este lenguaje de código puede ser incrustado en HTML5 puesto que es un código muy variable favorece el enlace entre la interfaz de usuario y los servidores. Diferenciar el lenguaje de programación PHP con los típicos como pueden ser HTML, CSS o JavaScript es sencillo, estos último son leídos primero por el navegador al momento de acceder a una web, mientras que PHP se ejecuta directamente en el servidor de la página.

  • Símbolos: PHP utiliza símbolos como: $ (Signo de dólar): Se utiliza para declarar y acceder a variables en PHP. Por ejemplo, $variable se utiliza para declarar una variable y $variable = 10 para asignarle un valor. [] (Corchetes): Se utilizan para acceder a elementos de matrices y arreglos asociativos. Por ejemplo, $array[0] accederá al primer elemento de un arreglo. foreach: Se utiliza para iterar sobre arreglos y colecciones de datos en PHP. require: Similar a include, se utiliza para incluir archivos en un script PHP, pero genera un error fatal si el archivo no se encuentra.

  • Sintaxis: La sintaxis de PHP se basa en etiquetas HTML con fragmentos de código PHP incrustados.

  • Esqueleto de programa: Los programas en PHP se componen de bloques de código PHP incrustados en documentos HTML.

  • Aplicaciones: Para editar y ejecutar PHP, se puede utilizar cualquier editor de texto y servidores web que admitan PHP, como Apache.

  • Paradigma: PHP es un lenguaje de programación de propósito general con enfoque en la programación web. Combina características de la programación imperativa y orientada a objetos.
